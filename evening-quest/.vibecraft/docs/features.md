# Evening Quest

## Цель проекта

Геймифицировать вечерние прогулки, превратив каждую из них в маленькое приключение-квест: пользователь выбирает точку на карте и идёт к ней с «миссией» (донести артефакт, кольцо, посылку), отслеживая прогресс в реальном времени через GPS и получая достижения.

## Основные возможности

### Выбор квеста

- **Цель:** Пользователь выбирает, куда сегодня отправиться.
- **Описание:** На карте отображаются ближайшие интересные места (кафе, парки, магазины, памятники). Пользователь выбирает одно из них или ставит произвольную метку — это становится целью квеста. Квесту присваивается случайное «легендарное» название (например, «Доставить кольцо в Одинокую Башню»).
- **Критерии приёмки:**
  - Карта отображается с возможностью приближения/удаления
  - Пользователь может выбрать предложенную точку или поставить свою метку
  - После выбора показывается расстояние до цели и примерное время в пути
  - Квест получает случайное тематическое название

### Отслеживание прогресса по GPS

- **Цель:** Пользователь видит, как продвигается к цели в реальном времени во время реальной прогулки.
- **Описание:** Когда пользователь начинает квест, приложение отслеживает его реальное местоположение через GPS. На карте отображается текущее местоположение, пройденный маршрут и оставшийся путь до цели. Показывается процент завершения, пройденное расстояние и оставшееся время. Когда пользователь оказывается в радиусе 50 метров от цели — квест автоматически завершается.
- **Критерии приёмки:**
  - Текущее местоположение обновляется на карте в реальном времени через GPS
  - Пройденный маршрут отрисовывается линией на карте
  - Отображается прогресс-бар с процентом завершения
  - Показывается пройденное расстояние и оставшееся до цели
  - При приближении к цели на 50 метров квест завершается автоматически

### Завершение квеста

- **Цель:** Получить удовлетворение от выполненной миссии.
- **Описание:** Когда пользователь достигает цели (в радиусе 50 м), показывается экран завершения: поздравление, название выполненного квеста, время в пути, пройденная дистанция. Квест сохраняется в историю.
- **Критерии приёмки:**
  - При достижении цели автоматически показывается экран завершения
  - На экране — название квеста, дистанция, время, тематическое поздравление
  - Квест сохраняется в историю

### История квестов

- **Цель:** Видеть все свои прошлые прогулки-приключения.
- **Описание:** Список всех завершённых квестов с датой, названием, дистанцией и временем. Можно посмотреть маршрут на карте.
- **Критерии приёмки:**
  - Отображается список всех завершённых квестов
  - Каждый элемент показывает дату, название, дистанцию, время
  - Можно открыть квест и посмотреть его маршрут на карте

### Достижения

- **Цель:** Получать награды за регулярность и рекорды.
- **Описание:** Система бейджей за первые прогулки, серии дней подряд, рекордную дистанцию. Бейджи отображаются в профиле.
- **Критерии приёмки:**
  - Бейджи выдаются автоматически при выполнении условий
  - Список всех бейджей отображается в профиле
  - У каждого бейджа есть название, иконка и описание условия получения

## Дополнительные заметки

- Стиль интерфейса: светлое фэнтези для взрослых — как иллюстрированный атлас сказочных земель, без мультяшности. Карта в стилизованном «сказочном» виде, тёплая цветовая гамма.
- Приложение для вечерних прогулок пешком.
- Названия квестов генерируются случайным образом в тематике фэнтези-похода.
- Отслеживание только через реальное GPS, без симуляции движения на карте.
- Статический сайт на чистом HTML/CSS/JS, хостится на GitHub Pages.
- Данные хранятся в localStorage браузера.
